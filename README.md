# 📚 Study Streak

Study Streak is a simple application designed to help users build consistent study habits by tracking daily study progress and maintaining a streak. The project focuses on encouraging productivity and discipline through a minimal and easy-to-use interface.

---

## 🚀 Features

- Track daily study activity
- Maintain a continuous study streak
- Simple and clean user interface
- Encourages consistent learning habits
- Lightweight and easy to use

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Git

---

```
Study-Streak/

```

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/your-username/Study-Streak.git
```

2. Open the project folder.

3. Run the project by opening `index.html` in a web browser.

---

## 💡 Usage

- Open the application in a browser.
- Record daily study sessions.
- Maintain and track your study streak.
- Use it to stay consistent with learning goals.
